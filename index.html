O texto acima foi feito utilizando a tag "p", ela serve para criarmos um parágrafo em nosso site, todos os parágrafos deste site estão com estilo de fundo azul escuro e borda arredondada azul claro (backgroundcolor, border e borderradius).

Esse parágrafo está sendo afetado pela tag "article", uma tag semântica para mostrar um trecho de importância no conteúdo, além disso ela está sendo afetada pelo efeito CSS de classe com tamanho de fonte de 30px

A lista ordenada acima foi feita utilizando a tag "ol" para determina-la como esse tipo de lista e a tag "li" para definir cada item desta lista.
