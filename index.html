O HTML é uma linguagem de marcação, serve para indicar para o navegador como a estrutura do meu site deve ser lida.


O texto acima foi feito utilizando a tag "p", ela serve para criarmos um parágrafo em nosso site

Esse parágrafo está sendo afetado pela tag "article", uma tag semântica para mostrar um trecho de importância no conteúdo

  • Cuidar com as armadilhas
  • É trabalhoso e precisa de determinação
  • É um mercado de trabalho amplo
  • Pode lhe fornecer salários altos
  • Este item da lista está em itálico pela tag "em", além desse feito, ela guia o navegador a dar destaque para este item, isso é chamado de semântica
  • Este item está em negrito pela tag "strong", também dá destaque ao item
  • Este item está afetado pela tag "span", serve para dar menos destaque a uma palavra ou de certa forma diferencia-la do resto do conteúdo através de atributos inseridos nela

A lista não-ordenada acima foi feita utilizando a tag "ul" para determina-la como esse tipo de lista e a tag "li" para definir cada item desta lista.
  1. Começamos definindo que é um arquivo HTML
  2. Depois configuramos a página pela tag HEAD
  3. Após isso definimos o corpo do site dentro do conteúdo da tag BODY

A lista ordenada acima foi feita utilizando a tag "ol" para determina-la como esse tipo de lista e a tag "li" para definir cada item desta lista.

Esta linha está sendo afetada pela tag "section", ela funciona como a DIV, mas é uma tag semântica, isso define esta linha como importante para o navegador

Esse é um exemplo da tag de ancoragem "a", clique nela para ver seu efeito
Esta área de âncora está sendo afetada pela tag "div", ela serve para separar um conteúdo e fazermos alterações específicas nele (geralmente visuais)

O conteúdo do parágrafo até a área de âncora está sendo afetado pela tag "main", que também é uma tag semântica, que define todo este conteúdo como o príncipal do site para o navegador

Temos a tag "aside" também, que é para indicar um conteúdo menos relacionado ao principal, como uma área de anúncios e etc
